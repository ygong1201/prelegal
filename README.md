# prelegal

---

A platform for drafting common legal agreements.

## Status

This project is currently in progress and is expected to be completed within 1 week (by 2026-05-18).

## Templates

`templates/` contains the seed dataset of legal agreement templates the system can adapt for users. The current set is sourced from [Common Paper](https://github.com/CommonPaper):

- Mutual NDA — Standard Terms and Cover Page
- Cloud Service Agreement (CSA)
- Software License Agreement
- Professional Services Agreement (PSA)
- Data Processing Agreement (DPA)
- Service Level Agreement (SLA)
- Design Partner Agreement
- Partnership Agreement
- Pilot Agreement
- Business Associate Agreement (BAA)
- AI Addendum

`catalog.json` at the project root indexes every template with its name, description, filename, and upstream source repo.

All templates are made available under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/) with attribution to Common Paper. See `templates/LICENSE.txt`.
